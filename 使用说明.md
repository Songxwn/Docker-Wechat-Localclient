# WeChat Selkies - 本地远程客户端 - 使用说明

本文档说明如何安装、配置和使用本客户端，连接 wechat-selkies 服务并在本机使用网页版微信/QQ。

---

## 一、客户端是什么

本程序是 **WeChat Selkies 的本地远程客户端**，用来：

- 保存多个 wechat-selkies 服务地址（如本地 Docker、公司服务器、家里 NAS 等）
- 一键在应用内或系统浏览器中打开对应网页版微信/QQ 界面
- **剪贴板共享**：与 Docker 内 wechat-selkies 共享剪贴板，本机复制的文字和图片可在网页版微信中直接粘贴发送
- **文件拖入上传**：将本机文件拖入窗口即可自动上传到 Docker 内部，便于在微信中发送文件
- 为每个连接单独设置「忽略 SSL」「用系统浏览器打开」等选项

**前提**：需要先在一台机器上部署好 [wechat-selkies](https://github.com/nickrunning/wechat-selkies) 服务（通常在 Linux/Docker 中），本客户端只是用来「连接并打开」该服务的网页界面。

---

## 二、运行环境

- **系统**：Windows 10 或 Windows 11
- **wechat-selkies 服务**：已在 Docker 或 Linux 服务器上运行，且本机能够访问其 HTTPS 地址（一般为 `https://IP或域名:3001`）

---

## 三、安装与启动

### 方式一：从源码运行（需安装 Node.js）

1. 安装 [Node.js](https://nodejs.org/)（建议 18 及以上版本）
2. 在项目目录打开命令行，执行：
   ```bash
   npm install
   npm start
   ```

### 方式二：使用安装包

1. 从发布页或构建目录获取安装包（如 `WeChat Selkies Setup x.x.x.exe`）
2. 双击安装，按提示完成安装后从开始菜单或桌面快捷方式启动

---

## 四、基本使用

### 1. 首次使用

- 首次启动时，若尚未添加任何连接，程序会预置一条「本地微信」连接，地址为 `https://localhost:3001`。
- 若你的 wechat-selkies 就在本机（如本机 Docker），可直接点击该连接的 **「连接」** 试用；否则请先 **「编辑」** 修改地址，或点击 **「+ 添加连接」** 新建一条。

### 2. 添加新连接

1. 点击右上角 **「+ 添加连接」**
2. 在弹窗中填写：
   - **名称**：便于识别的名称，如「公司服务器」「家里 NAS」
   - **服务地址**：wechat-selkies 的完整访问地址，例如：
     - 本机：`https://localhost:3001`
     - 局域网：`https://192.168.1.100:3001`
     - 远程：`https://your-server.com:3001`
3. 按需勾选：
   - **忽略 SSL 证书错误**：本地或自签名证书时建议勾选
   - **使用系统默认浏览器打开**：勾选后点击「连接」时用系统浏览器打开，不勾选则在客户端内新窗口打开
4. 点击 **「保存」**

### 3. 连接列表操作

每条连接右侧提供：

- **连接**：使用当前连接的地址和选项打开微信/QQ 页面
- **编辑**：修改名称、地址和上述两个选项
- **设为默认**：将该连接设为默认（仅非默认连接显示此按钮）
- **删除**：删除该连接（会二次确认）

### 4. 使用系统托盘（若已配置）

若程序提供了托盘图标，可最小化到托盘；双击托盘图标可重新打开主窗口。

## 五、wechat-selkies 服务地址说明

- 默认 HTTPS 端口一般为 **3001**，完整地址形如：`https://主机:3001`
- 若 wechat-selkies 部署在 **本机 Docker**，本机访问可用：`https://localhost:3001`
- 若在 **WSL2 或另一台电脑**，请将 `localhost` 改为实际 IP，例如：`https://192.168.x.x:3001`
- 若通过 **域名或反向代理** 访问，填写你实际使用的完整 URL 即可

wechat-selkies 的部署方法请参考其官方仓库：[nickrunning/wechat-selkies](https://github.com/nickrunning/wechat-selkies)。

## 六、常见问题

**Q：点击「连接」后无法打开或提示证书错误？**  
- 确认 wechat-selkies 服务已启动且端口（如 3001）已开放。  
- 在编辑该连接时勾选「忽略 SSL 证书错误」后保存，再重试。

**Q：想用 Chrome/Edge 等浏览器打开，而不是客户端内窗口？**  
- 编辑对应连接，勾选「使用系统默认浏览器打开」并保存，之后点击「连接」即会用系统默认浏览器打开。

**Q：如何删除或修改某条连接？**  
- 在该连接右侧点击「编辑」可修改；点击「删除」可删除（会提示确认）。

**Q：数据保存在哪里？**  
- 连接的配置保存在本机（由 Electron 的配置存储管理），仅限当前用户、当前电脑使用，不会上传到任何服务器。
