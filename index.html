    使用说明

    • 请先在 Linux / Docker 中部署 wechat-selkies，并确保端口 3001 可访问。
    • 若在本机通过 WSL2 或远程 Linux 运行，请将 localhost 改为对应 IP（如 https://192.168.x.x:3001）。
    • 剪贴板共享：本机复制的文字、图片可与 Docker 内微信共享，在网页版微信中可直接粘贴发送。
    • 文件拖入：将文件拖入微信窗口可自动上传到 Docker 内部，用于在聊天中发送文件。
